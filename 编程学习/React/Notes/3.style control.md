#### 组件基础样式方案
- React组件基础的样式控制有两种方式
- 内部的元素**一定要写成驼峰命名法**
	1. 行内样式
	
	``` html
	<div style={{color: 'red'}}>this is div</div>
	``` 


	2. class类名控制
		- .css文件
			.foo {
				color: red,
			}
		- App.js
			- 使用**className**指定css文件中的类样式选择器
		![image.png](https://iili.io/JYoEELQ.png)
span {

}

}
